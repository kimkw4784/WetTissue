## 프로젝트 핵심 특징 (Core Features)

* **프리미엄 모션 UX:** `GSAP`와 `ScrollTrigger`를 활용하여 스크롤에 반응하는 정교한 인터랙션 구현 (Hero 섹션의 물티슈 분리 모션, 원단 생분해 타임라인 등).
* **부드러운 주행감:** `Lenis` 라이브러리를 바인딩하여 브라우저 기본 스크롤에 매끄러운 관성(Smooth Scroll)을 부여.
* **컴포넌트 중심 설계:** React 기반의 모듈화된 섹션 관리 및 확장 가능한 CSS 변수(Variables) 시스템 채택.
* **반응형 최적화:** 768px 브레이크포인트를 기준으로 모바일 환경에서는 모션을 간소화하고 네이티브 터치를 보장하여 퍼포먼스 최적화.

## 기술 스택 (Tech Stack)

### Frontend
* **Framework:** React 18+
* **Build Tool:** Vite
* **Styling:** Vanilla CSS (CSS Modules 패턴)

### Animation & Interaction
* **GSAP** (GreenSock Animation Platform) & **ScrollTrigger**
* **Lenis** (Smooth Scroll Management)
* **Lucide React** (Icon Pack)
